## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CallPage render ← CallContext | isMuted, isCameraOff, iceState consumed as read-only values from context |
| UserListPage Cancel button → hangUp() | Triggers teardown + call-end signal; same trust level as existing End Call button |
| Timer interval | Local setInterval in hook; no network I/O; only increments a local counter |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04 | Tampering | peerUsername in peer name overlay | accept | peerUsername is set from STOMP signal msg.from, overwritten server-side with JWT principal. Cannot be attacker-controlled. React JSX auto-escapes text content. |
| T-05-05 | Denial of Service | Timer interval accumulation | mitigate | useCallTimer returns clearInterval(id) from useEffect cleanup; prevents interval accumulation on StrictMode double-mount (Pitfall 4). |
| T-05-06 | Tampering | Unmount cleanup hangUp() call | accept | hangUp() is idempotent-safe in teardown (teardown only acts if pcRef/localStreamRef are non-null). Calling it on unmount with callStatus=idle is a no-op. |
| T-05-07 | Information Disclosure | ICE state label in UI | accept | ICE state is browser-controlled enum; only reveals connection quality to the local user. No sensitive data. |
| T-05-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed in Phase 5. Package legitimacy gate not required. |
